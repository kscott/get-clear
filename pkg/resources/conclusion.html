Almost there

Get Clear is installed. Two steps to finish:

  1. Connect mail
    Go to Fastmail → Settings → Privacy & Security → API tokens, create a new token with JMAP scope, then open Terminal and run:
    mail setup <your-token>
  2. Name your calendars
    Run calendar calendars to see your calendar names, then create
    ~/.config/calendar-cli/config.toml to define named groups like work and personal.

Each tool will ask for macOS permissions on first use — just approve them:

reminders list  ·  calendar today  ·  contacts lists  ·  sms open

Full guide: github.com/kscott/get-clear
